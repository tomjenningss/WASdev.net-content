IBM has recently made available a public beta for the IBM Java SDK version 9 for Linux. We've been testing Liberty on early access Java 9 drivers for some time. If you're curious to see our progress, you can read about it in these other blog posts: Part 1: A first look at Liberty on Java 9 Part 2: A first look at Liberty on Java 9 If you want to get your hands on IBM Java SDK version 9 and play with it yourself, you can get it from DockerHub or DeveloperWorks (you'll need to log in). If you are looking for Open JDK early access, you can get it from java.net.
